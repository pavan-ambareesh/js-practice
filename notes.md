Difference between var, const and let
all the above mentioned are to define a variable to hold data with some unique properties, 
    var keyword eanbles the user to use the variable anywhere in the entire JS.
    let keyword also does the same but the visibility is constrained inside a function only
    const keyword also stores the data inside a variable however that data cannot be overridden but it can be reinitialised

What is Variable shadowing
    variable shadowing is will enable the script to have the local var to have the same name as a global var but inside a function, the date/value stored in global var will remain untouched, when we are using the local var inside a function
